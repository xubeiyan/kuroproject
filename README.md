# kuroproject

一个黄油存档备份系统

## 介绍

就是一个简单的黄油存档备份系统     

## 运行环境

* Windows 10
* Windows 8.1
* Windows 7

开发者目前只能保证在Windows 8.1和Windows 10上面进行测试。由于没有Windows 7的设备（其实是懒得装），所以暂时无法保证本软件在Windows 7上运行的正确性和可靠性。如有问题，请到[这里](https://github.com/xubeiyan/kuroproject/issues)提交（其实根本不看）。

Windows 10可以直接运行，使用Windows 8或者之前版本的Windows，请安装[.NET Framework 4.7](https://www.microsoft.com/zh-CN/download/details.aspx?id=55167)。
